## Features of Each Pipeline

Each algorithm pipeline includes:

-   Dataset loading
-   Missing value handling
-   User-defined column dropping
-   Categorical encoding
-   Numeric feature scaling
-   Hyperparameter experimentation
-   Performance metrics (human-readable)
-   Data visualization
-   Modular model class design

Clustering pipelines additionally include:

-   PCA-based cluster visualization
-   Dendrogram plotting (Hierarchical)
